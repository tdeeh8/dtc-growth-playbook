
## Triage-first audits (v4.1)

Two additional audit skills optimized for a triage-first workflow (score each platform RED/YELLOW/GREEN first, deep-dive only where problems exist).

| Skill | Data source | Command |
|---|---|---|
| `ads-audit` | Adzviser MCP | `/ads-audit [client]` |
| `databox-audit` | Databox MCP | `/databox-audit [client]` |

Both cover: Google Ads, Meta Ads, Amazon Ads, GA4, Shopify/BigCommerce. Neither covers SEO, CRO, or Klaviyo — use `/audit` for those.
