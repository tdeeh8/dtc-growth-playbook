# CRO Checklist — Page-by-Page Conversion Element Audit

Use this checklist during the site-audit-v2 walkthrough. Score each element as **Strong**, **OK**, **Weak**, or **Missing**. Record specific observations — not just the score.

---

## Homepage

### Above the Fold (First 3 Seconds)

| Element | What to Check | Strong | Weak |
|---------|--------------|--------|------|
| Value proposition | Can a new visitor understand what you sell + why in <3s? | Clear, specific, benefit-driven headline | Vague, generic, or buried below fold |
| Hero image/video | Does the visual support the value prop and show the product in use? | Lifestyle imagery showing product in context | Stock photo, no product, or purely decorative |
| Primary CTA | Does it drive toward shopping (Shop Now, Browse Collection)? | Single clear CTA above fold driving to products | Multiple competing CTAs, or CTA = "Learn More" / "Our Story" |
| Navigation visibility | Can users immediately find product categories? | Clear category nav, visible without hovering | Categories buried in hamburger menu on desktop, or unclear labels |
| Search bar | Is search prominent and easy to find? | Visible in header, icon + text field | Hidden, icon-only, or missing entirely |
| Announcement bar | Is there a site-wide promo or shipping threshold? | Clear offer (free shipping over $X, X% off) | No bar, or bar with irrelevant info |

### Below the Fold (Scroll Content)

| Element | What to Check | Strong | Weak |
|---------|--------------|--------|------|
| Content hierarchy | Does the page guide users toward products/collections? | Logical flow: hero > featured products > categories > social proof | Random content blocks with no clear path |
| Social proof | Reviews, press logos, UGC, testimonials | Multiple proof types near key decision points | No social proof, or only buried on a separate page |
| Featured products | Are bestsellers or key products showcased? | Curated selection with prices, reviews, quick-add | No featured products on homepage |
| Collection links | Easy paths to browse product categories? | Visual category blocks with clear labels | Text-only links buried in footer |
| Email/SMS capture | Is there a non-intrusive opt-in opportunity? | Embedded form or well-timed popup with clear value exchange | Immediate popup blocking content, or no capture at all |
| Footer completeness | Contact, policies, trust signals | Contact info, return policy, shipping info, payment icons, social links | Minimal footer missing key trust elements |

### Popups & Interruptions

| Element | What to Check | Strong | Weak |
|---------|--------------|--------|------|
| Popup timing | When does the first popup appear? | 15-30s delay or exit-intent trigger | Immediate (0-5s) — blocks first impression |
| Popup stacking | Do multiple overlays compete for attention? | Single popup at a time, no overlap with chat/cookie banners | 2+ overlays visible simultaneously |
| Popup offer | Is the incentive compelling? | Clear value exchange (% off, free shipping, exclusive access) | Generic "join our newsletter" with no incentive |
| Dismiss UX | Easy to close? | Obvious X button, clicking outside closes it | Hard to find close button, dismissing triggers another popup |
| Chat widget | Helpful or obstructive? | Minimized by default, doesn't overlap key elements | Covers CTA buttons, auto-opens with noise |

## Collection / Category Pages

| Element | What to Check | Strong | Weak |
|---------|--------------|--------|------|
| Product grid layout | Is the grid clean and scannable? | Consistent card sizes, 3-4 per row desktop, adequate whitespace | Mismatched images, cluttered layout, too many/few per row |
| Product card info | What info is visible before clicking in? | Image + title + price + review stars + quick-add | Image + title only — must click to see price or reviews |
| Image quality | Are product images high quality and consistent? | Professional, consistent lighting/angle, lifestyle mix | Low-res, inconsistent backgrounds, missing images |
| Hover states | Do product cards have useful hover behavior? | Secondary image on hover, quick-add button appears | No hover state, or broken/glitchy hover |
| Filtering | Can users filter by relevant attributes? | Size, color, price, material, availability — relevant to product type | No filters, or irrelevant filter options |
| Sorting | Can users sort meaningfully? | Price (low-high, high-low), bestselling, newest, review rating | No sort options, or only alphabetical |
| Pagination | How are additional products loaded? | Infinite scroll with "load more" or numbered pages with product count | No indication of total products, or broken pagination |
| Collection description | Is there contextual content for SEO and user orientation? | Brief, useful description at top explaining the collection | Missing entirely, or 500-word keyword-stuffed block |
| Breadcrumbs | Can users orient themselves in the site hierarchy? | Full breadcrumb trail (Home > Category > Subcategory) | No breadcrumbs — back button is the only option |
| Empty state | What happens when filters return 0 results? | Helpful message with suggestions to broaden search | Blank page or generic "no results" with no guidance |
| Out of stock handling | How are OOS products displayed? | Labeled clearly, allow back-in-stock notification, deprioritized in grid | Mixed in with available products with no indication |

## Product Pages (PDP)

### Product Information

| Element | What to Check | Strong | Weak |
|---------|--------------|--------|------|
| Images | Quantity, quality, variety | 5+ images: hero, lifestyle, detail shots, scale reference, video | 1-2 low-quality images, no context |
| Image zoom/gallery | Can users inspect details? | Zoom on hover/click, swipeable gallery, fullscreen option | No zoom, broken gallery, tiny thumbnails |
| Video | Is there product video? | Demo or lifestyle video on PDP | No video (not always a problem, but noted) |
| Title | Clear, descriptive product name | Includes key attributes (material, size, use case) | Generic name that doesn't differentiate |
| Price | Clear and prominent | Current price near title, sale price with strikethrough original | Price hidden, confusing discount display, or "from $X" without clarity |
| Product description | Does it sell, not just describe? | Benefits + features, scannable format, addresses objections | Wall of text, specs-only, or copied from manufacturer |
| Specifications | Are technical details available? | Organized specs section (dimensions, materials, weight, care) | No specs, or buried in the description paragraph |
| Variant selection | Color/size/option selectors | Visual swatches, clear labels, selected state obvious, OOS variants marked | Text-only dropdowns, no indication of availability per variant |

### Trust & Conversion Elements

| Element | What to Check | Strong | Weak |
|---------|--------------|--------|------|
| Reviews | On-page, visible, credible | Star rating near title + review count + full reviews section with photos | No reviews, or reviews on a separate page only |
| Review quality | Are reviews detailed and authentic? | Verified purchases, photo reviews, varied ratings | All 5-star, generic text, or clearly fake |
| Add-to-cart button | Prominent and accessible | High-contrast, above fold, sticky on scroll (mobile), clear label | Below fold, low contrast, generic "Submit" label |
| Shipping info | Transparent on PDP | "Free shipping over $X" or flat rate shown. Estimated delivery date | "Calculated at checkout" or no mention at all |
| Return policy | Visible or linked from PDP | Policy summary near CTA or clear link | Policy only in footer or missing entirely |
| Trust badges | Near the CTA | Secure checkout, money-back guarantee, payment icons, warranty | No trust signals near the purchase decision point |
| BNPL / Financing | For AOV $100+ | Affirm/Klarna/Afterpay callout near price with monthly payment preview | Not offered, or only visible at checkout (too late) |
| Stock/urgency | Appropriate urgency signals | Low stock indicator, estimated delivery, "X people viewing" (if real) | Fake urgency ("Only 2 left!" on every product) or no info |
| Social proof on PDP | Beyond reviews | "X customers bought this", UGC gallery, "as featured in" | Reviews are the only social proof |
| Cross-sells | Related product recommendations | "Frequently bought together" with add-to-cart, "Complete the look" | No recommendations, or irrelevant suggestions |

## Cart & Checkout

### Cart (Page or Drawer)

| Element | What to Check | Strong | Weak |
|---------|--------------|--------|------|
| Cart access | How do users reach the cart? | Slide-out drawer on add-to-cart + persistent cart icon with item count | Separate page only, no item count indicator |
| Product details in cart | Is the item clearly identified? | Image, name, variant, unit price, quantity editor, line total | Name-only, no image, can't edit quantity |
| Subtotal | Are costs transparent? | Subtotal + estimated tax + shipping shown (or "Free" if applicable) | "Calculated at checkout" for taxes and shipping |
| Free shipping bar | If threshold exists, is progress shown? | "You're $X away from free shipping!" with progress indicator | Threshold mentioned on homepage but not in cart |
| Cart cross-sell | Last chance to increase AOV | 1-3 relevant add-ons (accessories, bundles, warranties) | No upsell, or irrelevant suggestions cluttering the cart |
| Express checkout | Accelerated payment options | Shop Pay, Apple Pay, Google Pay, PayPal above the main CTA | Express options only at final checkout step |
| Cart edit | Can users easily modify their cart? | Remove, quantity +/-, variant change without leaving cart | Must go back to PDP to change anything |
| Empty cart | What happens with an empty cart? | Helpful message + link to bestsellers or collections | Blank page or generic "Your cart is empty" |

### Checkout

| Element | What to Check | Strong | Weak |
|---------|--------------|--------|------|
| Guest checkout | Can users buy without creating an account? | Guest checkout prominent, account creation optional post-purchase | Forced account creation before purchase |
| Form UX | Are forms optimized? | Minimal fields, autofill-friendly, address validation, inline error messages | Too many fields, no autofill, errors only on submit |
| Order summary | Is it always visible? | Persistent order summary with images throughout checkout | Summary hidden or collapsed, must click to expand |
| Payment options | Breadth and visibility | Credit card + Shop Pay + PayPal + Apple Pay + BNPL | Credit card only, or options hidden behind tabs |
| Shipping options | Clear and honest | Named options with prices and estimated delivery dates | One option, no delivery estimate, surprise costs |
| Discount code field | Present but appropriate | Collapsed/subtle input that doesn't encourage cart abandonment | Prominently displayed, screaming "go find a coupon code" |
| Trust signals in checkout | Reassurance at commitment moment | Security badge, guarantee reminder, "30-day returns" near pay button | No trust elements in checkout |
| Error handling | How are errors communicated? | Inline validation, clear error messages, field highlighting | Generic "an error occurred" with no specifics |
| Order confirmation | What happens post-purchase? | Clear confirmation page + email, cross-sell opportunity, account creation offer | Minimal confirmation, no email, dead end |

## Post-Purchase (Observable from Front-End)

| Element | What to Check | Strong | Weak |
|---------|--------------|--------|------|
| Confirmation page | Does it maximize the post-purchase moment? | Order details + cross-sell + referral program + account creation | Bare "thank you" page |
| Email capture | Is the buyer opted into marketing? | Checkbox during checkout (pre-checked where legal) or post-purchase prompt | No opt-in mechanism visible |
| Referral program | Is there a word-of-mouth mechanism? | Post-purchase referral offer ("Give $10, get $10") | No referral program visible |
| Loyalty program | Is there a visible loyalty/rewards program? | Points earned on purchase, visible program benefits | No loyalty program, or only accessible from footer link |

## Site-Wide Elements

| Element | What to Check | Strong | Weak |
|---------|--------------|--------|------|
| Page load speed | Perceived performance | Pages load in <2s, no visible layout shifts, images load progressively | Noticeable lag, layout shifts, slow image loading |
| 404 page | Is the error page helpful? | Branded 404 with search bar, popular links, and nav | Default server 404 or blank page |
| Search functionality | Does site search work well? | Autocomplete, relevant results, handles typos, filters within results | No autocomplete, irrelevant results, broken for common queries |
| Accessibility basics | Can all users navigate? | Sufficient contrast, alt text on images, keyboard navigable | Low contrast text, missing alt text, elements only accessible via mouse |
| SSL/Security | Is the site secure? | Valid SSL certificate, no mixed content warnings | SSL issues, browser security warnings |
| Cookie consent | GDPR/CCPA compliance | Unobtrusive banner with clear opt-in/out | Missing, or massive overlay blocking the page |
