# MAT Testing (Messaging Architecture Testing)

Tests different value propositions against audience segments to find which message drives the most engagement and conversions. Testing the reason someone should buy, not the ad itself.

Example VPs: cost savings, convenience, quality/craftsmanship, status/identity, health/wellness.

## How to Run

- Identify 3-5 VPs from customer research, reviews, competitor positioning
- Under Andromeda: load all VP variations as different creatives into ONE ad set with broad targeting. Do NOT split into separate ad sets per VP.
- Minimum $150 per test variation
- Run 7-14 days for statistical significance
- The VP that wins on engagement AND intent signals is your lead message

## Campaign Type for MAT

- Budget available ($150+/variation): ATC-optimized campaign. Real intent signal, cleaner data, translates to scaling.
- Budget tight or new account with no pixel data: ThruPlay campaign. Cheaper, faster to significance. Validate winner with short ATC/purchase test before scaling.
- Do NOT use landing page views. Low bar, weak quality signal.

Key rule: match the MAT test event to your scaling event. If scaling on purchase, test on ATC. If scaling on Layer 1 reach, test on ThruPlay. Testing on one event and scaling on another is unreliable.

## Cadence

Refresh quarterly, when entering a new season, or when launching new products. Not a one-time exercise.
