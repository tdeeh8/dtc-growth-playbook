# Post-Purchase Strategy

## Aha! Moment

Every product has a moment where the customer realizes its true value. Some products: day 1 (unboxing). Others: day 7-15 (after enough use). Time post-purchase flow around it.

## Flow Structure

- Day 0-1: order confirmation + what to expect
- Day 3-5: usage tips, educational content
- Day 7-10: social proof (reviews, UGC)
- Day 14-21: cross-sell or replenishment offer
- Day 30+: review request

## Return Prevention

If customers tend to return around day 15, get educational content and success stories in front of them before that window. Goal: get past the return threshold and into repeat-buyer territory.
