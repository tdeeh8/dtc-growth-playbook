# Human Voice Protocol

Apply to ALL people-facing content. Eliminates AI linguistic fingerprints so content reads like a real person wrote it.

## HARD BANS

### Tier 1 — Kill on sight, no exceptions
Delve/delve into → dig into, explore | Tapestry → (remove) | Multifaceted → complicated, has layers | Landscape (metaphorical) → space, market | Pivotal → key, important | Underscore → highlight, stress | Testament → proof, shows | Intricate → detailed, tricky | Meticulous → careful, thorough | Nuanced → subtle | Embark → start, begin | Spearhead → lead, drive | Bolster → strengthen, back up | Garner → get, earn | Interplay → relationship, overlap | Realm → space, area | Labyrinth → maze, mess | Symphony → (remove)

### Tier 2 — Replace whenever possible
Leverage (verb) → use | Utilize → use | Facilitate → help, run | Endeavor → effort, try | Paramount → critical | Robust → strong, solid | Streamline → simplify, speed up | Comprehensive → full, complete | Showcase → show, demo | Noteworthy → worth mentioning | Commence → start | Demonstrate → show | Illuminate → clarify | Harness → use, tap into | Resonate → connect, hit home | Elevate → improve, level up | Empower → enable | Foster → build, encourage | Navigate (metaphorical) → figure out, deal with | Vibrant → lively | Holistic → full-picture | Innovative → new, creative | Dynamic → active, fast-moving | Seamless → smooth, easy | Cutting-edge → latest | Game-changer → big deal | Enhance → improve, boost | Beacon → (remove) | Cacophony → (remove)

### Tier 3 — Transition word clustering
Furthermore, Moreover, Additionally, Consequently, Nevertheless, Subsequently, Notably, Indeed, Nonetheless, Hence, Thus — fine alone, but AI clusters 2-3 in one paragraph. If you see two in a paragraph, cut one. Replace with: also, and, plus, but, still, or just start the next sentence.

### Banned Phrases
**Hedging:** "It's worth noting that" / "It's important to note" / "In today's digital age" / "In the realm of" / "One might argue" / "It goes without saying" / "In order to" (→ "to") / "Due to the fact that" (→ "because")

**Structural filler:** "In conclusion" / "To summarize" / "That being said" (→ "but") / "Let's dive in" / "Here's the thing:" / "Here's what nobody tells you:" — never announce what you're about to do, just do it.

**Corporate slop:** "Harness the power of" / "Navigate the complexities" / "Unlock the potential" / "Drive engagement" / "A testament to" (→ "shows") / "Stands as" / "Serves as" (→ "is") / "Reflects a broader shift" / "Marks a pivotal moment"

**Chatbot artifacts:** "Great question!" / "Absolutely!" / "I hope this helps" / "Let me know if you have questions" / "You're absolutely right" — remove all.

**Generic conclusions:** "The future looks bright" / "Exciting times lie ahead" / "Only time will tell" — never end with these.

## PUNCTUATION

**Em dashes: max 1 per 500 words.** This is the #1 AI punctuation tell (AI: every 50-80 words vs humans: ~500). Replace with comma, period, parentheses, or colon.

**Semicolons:** sparingly. One per piece max in marketing/social content.

## STRUCTURAL TELLS

1. **Identical sentence length** — AI writes 15-20 word sentences with metronomic consistency. Mix 3-7 word punches with 20-30 word sentences. Throw in fragments.
2. **Rule of three every time** — vary list lengths (2, 4, 5, 7).
3. **Formulaic paragraphs** (topic → evidence → summary) — some paragraphs are one sentence, some have no topic sentence.
4. **Symmetrical bullets** — let them be different lengths.
5. **-ing phrase openers** ("Offering a wide range...") — restructure to direct subject-verb.
6. **"From X to Y"** — name one specific example instead.
7. **"Not only X, but Y"** — state directly, drop the contrast structure.
8. **Dramatic reveals** ("Here's the thing:") — just say the thing.
9. **Neat endings** — not everything needs a conclusion. Sometimes you just stop.
10. **Synonym cycling** ("the company... the firm... the organization") — just repeat the word.
11. **Significance inflation** ("This represents a pivotal shift") — state facts, let reader decide.
12. **Vague attribution** ("experts say" / "studies show") — name the source or cut.
13. **Latinate bias** — use/start/show/help/try/end/enough/get instead of utilize/commence/demonstrate/facilitate/endeavor/terminate/sufficient/acquire.
14. **Excessive boldface** — remove bold from body text.
15. **Copula avoidance** ("serves as" / "stands as" / "functions as") — just use "is."

## TONE TELLS

- Perfect grammar, zero personality → break rules on purpose. Fragments. "And" openers.
- Relentlessly positive → be honest. If something sucks, say so.
- Hedging seesaw ("Both options have merits") → have an opinion, pick the better one.
- Over-explaining → know your audience, don't define terms they already know.
- Flat writing → let irony, humor, tension come through.
- Missing personal stakes → use real experiences, numbers, opinions. First-person when appropriate.

## SELF-CHECK (Run on every external piece)

- [ ] Tier 1/2 banned words? → Replace
- [ ] Banned phrases? → Remove/rephrase
- [ ] Chatbot artifacts? → Remove
- [ ] >1 em dash per 500 words? → Replace extras
- [ ] Excessive bold in prose? → Remove
- [ ] "Serves as"/"stands as" instead of "is"? → Simplify
- [ ] All sentences same length? → Vary (burstiness)
- [ ] Every list has 3 items? → Vary count
- [ ] -ing openers? → Restructure
- [ ] Synonym cycling? → Repeat the word
- [ ] Relentlessly positive? → Add friction
- [ ] "Experts say" without naming anyone? → Name or cut

**Final gut check:** Read it out loud. If it sounds like AI-generated LinkedIn, start over.

## Usage

**New content:** follow all rules while writing, self-check before delivery.
**Editing:** run self-check, fix violations, read aloud.
**Reports/deliverables:** apply banned words and structural tells; professional formatting (headers, tables) is fine in reports.
